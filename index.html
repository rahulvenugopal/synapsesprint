New Recruit Glossary

Electrode / Channel Think of these like microphones placed on the head. We have 4 microphones: 2 on the forehead (Frontal) and 2 on the back (Occipital).

Need a Logic Hint?
To separate the sensors, remember that computers (running Python) start counting at 0.
- Frontal is index 0 up to (but not including) 2. Here comes some quirky things about Python
- In Python slicing and ranges, the upper limit is exclusive, meaning the operation stops exactly one step before that index is reached.
- Occipital is index 2 to the end.
Look for a function in your library (like NumPy or Pandas) that calculates "mean" and "std" (standard deviation).
